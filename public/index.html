الإعدادات العامة

Google Sheets — النسخ الاحتياطي التلقائي

خطوات الإعداد:

  1. أنشئ ملف Google Sheet جديد
  2. من القائمة: Extensions → Apps Script
  3. احذف الكود الموجود والصق كود Apps Script الموجود في js/sheets.js
  4. اضغط Deploy → New deployment → Web app
  5. اختر: Execute as: Me — Who has access: Anyone
  6. انسخ الـ URL وضعه في الحقل أدناه
واتساب تلقائي — مسح QR وإرسال فوري ● جارٍ التشغيل...

جارٍ فحص السيرفر...

خطوات التشغيل:

  1. انقر نقرتين على start.bat لتشغيل النظام
  2. سيفتح المتصفح تلقائياً على http://localhost:3001
  3. سجّل الدخول ثم افتح الإعدادات ← واتساب
  4. امسح QR Code الذي يظهر هنا بهاتفك من:
    واتساب ← الأجهزة المرتبطة ← ربط جهاز
  5. بعد الربط، التذكيرات ستُرسل تلقائياً لكل العملاء ✅

البوت يبقى متصلاً طالما start.bat يعمل — يُرسِل لجميع العملاء قبل يوم أو 3 أيام من انتهاء الاشتراك.
